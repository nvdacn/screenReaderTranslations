Enkelttegns navigering

Følgende taster flytter til det næste element, ved at tilføje skift-tasten flyttes til det forrige element:

  • H: overskrift
  • L: liste
  • I: listeelement
  • T: tabel
  • K: link
  • N: tekst som ikke er et link
  • F: formularfelt
  • U: ubesøgt link
  • V: besøgt link
  • E: redigeringsfelt
  • B: knap
  • X: afkrydsningsfelt
  • C: komboboks
  • R: radioknap
  • Q: blok citat
  • S: separator
  • M: ramme
  • G: grafik
  • D: landmærke
  • O: indlejret objekt
  • 1 til 6: overskrifter på niveau 1 til 6

Konfigurering af NVDA

Indstillinger

Stemeindstillinger (NVDA+control+v)

Name Desktop key Laptop key Description
Oplæs alle sætningstegn NVDA+p NVDA+p Når denne indstilling er slået til, læser NVDA alle sætningstegn som ord. Dette er især nyttigt ved korrekturlæsning.

Runden talesyntese-indstillinger

Name Desktop key Laptop key Description
Flyt til næste talesynteseindstilling NVDA+control+rightArrow NVDA+control+rightArrow Flytter til den næste tilgængeligetalesyntese-indstilling før den aktuelle og folder rundt til den første indstilling efter den sidste
Flyt til forrige talesynteseindstilling NVDA+control+leftArrow NVDA+control+leftArrow Flytter til den forrige tilgængeligetalesyntese-indstilling før den aktuelle og folder rundt til den sidste indstilling efter den første
Forøg aktuelle synteseindstilling NVDA+control+upArrow NVDA+control+upArrow forøger den synteseindstilling, som du i øjeblikket er ved at ændre fx forøger hastigheden, vælger den næste stemme, forøger lydstyrken
Formindsk aktuelle synteseindstilling NVDA+control+ downArrow NVDA+control+downArrow formindsker den synteseindstilling, som du i øjeblikket er ved at ændre fx formindsker hastigheden, vælger den forrige stemme, formindsker lydstyrken

Braille-indstillinger

Name Desktop key Laptop key Description
Braille følger NVDA+control+t NVDA+control+t Denne indstilling giver mulighed for at bestemme, om punktdisplayet skal følge systemfokus, eller om det skal følge navigatorobjektet / den ubegrænsede markør.

Tastatur-indstillinger (NVDA+control+k)

Name Desktop key Laptop key Description
Oplæs indtastede tegn NVDA+2 NVDA+2 Når denne funktion er slået til, oplæser NVDA alle de tegn, du skriver på tastaturet.
Oplæs indtastet ord NVDA+3 NVDA+3 Når denne funktion er slået til, oplæser NVDA alle de ord, du skriver på tastaturet.
Oplæs funktionstaster NVDA+4 NVDA+4 Nårdenne funktion er slået til, oplæser NVDA alle taster på tastaturet, der ikke er bogstaver, når du trykker på dem. Dette omfatter ligeledes tastekombinationer som control plus et bogstav.

Indstillinger for musen (NVDA+control+m)

Name Desktop key Laptop key Description
Slå musefølgning til NVDA+m NVDA+shift+m Når denne funktion er slået til, oplæser NVDA den tekst, der i øjeblikket befinder sig under musemarkøren i takt med, at musemarkøren flyttes rundt på skærmen. Det giver mulighed for at finde ting på skærmen ved fysisk at bevæge musen, i stedet for at lede efter dem ved hjælp af objektnavigation.

Indstillinger for den ubegrænsede markør

Name Desktop key Laptop key Description
Følg systemfokus NVDA+7 NVDA+control+7 Når denne funktion er slået til, placeres den ubegrænsede markør altid i samme objekt som det aktuelle systemfokus, når det ændrer sig.
Følg systemmarkøren NVDA+6 NVDA+6 Når denne funktion er slået til, flyttes den ubegrænsede markør automatisk til systemmarkørens position, hver gang den flytter sig.

Objekt-præsentations indstillinger (NVDA+control+o)

Name Desktop key Laptop key Description
Output for behandlingslinje NVDA+u NVDA+control+f2 Denne indstilling bestemmer, hvordan NVDA annoncerer opdateringer af behandlingslinjer.

Indstillinger for virtuelle buffers (NVDA+control+b)

Name Desktop key Laptop key Description
Brug skærmlayout NVDA+v NVDA+v Denne indstilling giver dig mulighed for at bestemme, om indhold i den virtuelle buffer skal præsentere indhold som links og andre felter på en selvstændig linje, eller om indholdet skal beholdes i selve tekstens rækkefølge på samme måde, som det vises visuelt.
