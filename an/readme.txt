NonVisual Desktop Access (NVDA)
URL: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) ye un lector de pantalla de código ubierto (y libre) t'o Sistema Operativo Windows. Iste programa puede capacitar a la chent ciega u deficient visual l'acceder a ordinadors executando Windows por un coste no mayor que una persona vidente. NVDA permite a l'usuario conoixér qué ye ocurrindo en a pantalla preguntando a o Sistema Operativo y utilizando  un sintetizador de voz t'a información de salida.

Ta mas información, por favor mira la guida de l'usuario incluída. Pueden estar trobaus información adicional y recursos en o Puesto Web de NVDA.
