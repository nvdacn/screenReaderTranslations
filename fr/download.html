Voix Vocalizer pour NVDA

Bienvenue sur la page de téléchargement des voix Vocalizer pour NVDA!

Ici vous pouvez télécharger les installateurs pour le pilote de Vocalizer et les différentes voix.

Pour installer le driver et les voix désirées, procédez comme suit :

  1. Ouvrez le menu de NVDA, avec NVDA+N;
  2. Descendez jusqu'au sous-menu Outils et appuyez sur la flèche droite pour l'ouvrir;
  3. Trouvez l'élément "Gestion des greffons" et appuyez sur Entrée ;
  4. Cliquez sur le bouton "Installer" et dans la boîte de dialogue parcourez votre disque dure pour trouver le greffon désiré ;
  5. Après avoir cliquer sur le bouton "Ouvrir" une boîte de dialogue d'avertissement vous demandera de confirmer l'installation. Cliquez sur Oui ;
  6. Faites ceci jusqu'à ce que vous ayez installé tous les greffons désirés. Pour finir, Tabulez jusqu'au bouton "Fermer" et appuyez sur Entrée. Redémarrez NVDA lorsque demandé.
