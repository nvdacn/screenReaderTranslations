So this means that if you quickly need to disable both beeps en announcing of objects, then just press NVDA+m. De beeps are also louder or softer depending on how bright het scherm is at that point.

Object presentatie en interaction

  • Improved support voor most common treeview controls. NVDA nu tells you how many items are in de branch when you expand it. It also announces de level when moving in en out of branches. And, it announces de current item number en number of items, according to de current branch, not de entire treeview.
  • Improved what is announced when focus changes as you move around applicaties or de operating system. Nu instead of just hearing de control you land on, you hear informatie about any controls this control is positioned inside of. For instance if you tab en land on een knop inside een groupbox, de groupbox will also get announced.
  • NVDA nu tries to speak de bericht inside many dialog boxes as they appear. This is accurate most of de time, though there are still many dialogs that arn't as good as they could be.
  • Added een report object descriptions checkbox to de object presentatie settings dialog. Power users may wish to sometimes uncheck this to stop NVDA announcing een lot of extra descriptions on particular controls, such as in Java applicaties.
  • NVDA automatisch announces selected text in edit controls when focus moves to them. If there isn't any selected text, then it just announces de current line like usual.
  • NVDA is een lot more careful nu when it plays beeps to indicate progress bar changes in applicaties. It niet langer goes crazy in Eclipse applicaties such as Lotus Notes/Symphony, en Accessibility Probe.
