Voces de Vocalizer para o NVDA

¡Benvido á páxina de descarga do Vocalizer para NVDA!

Aquí podes descargar un complemento para o NVDA, co controlador Vocalizer e varias voces dispoñibles.

Para instalar o controlador e as voces que decidas, fai os seguintes pasos:

  1. Abre o menú NVDA, con NVDA+N;
  2. Preme flecha abaixo ate o submenú Ferramentas e preme Intro para abrilo;
  3. Procura o elemento "Administrador de Complementos" e preme Intro;
  4. Fai clic sobor do botón "Instalar" e no seguinte diálogo examina para localizar o Complemento desexado;
  5. Despois de premer o botón "Abrir", NVDA presentará un diálogo de aviso preguntándoche para confirmar que queres instalar. Responde sí para continuar;
  6. Executa estos pasos ate ter instalados todos os Complementos desexados. Para rematar, Tabula ate o botón "Pechar" e preme Intro. Reinicia o NVDA cando se che indique.
