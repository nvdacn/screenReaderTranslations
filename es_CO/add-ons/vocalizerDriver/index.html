Voces Vocalizer para NVDA

¡Bienvenido a Vocalizer para NVDA!

Tiflotecnia, Lda., se complace en anunciar la disponibilidad de un nuevo sintetizador de alta calidad para NVDA, ¡el mejor lector de pantalla de código abierto del Mundo!

Este nuevo sintetizador, el Vocalizer bien conocido de Nuance, utilizando un controlador específico para NVDA, desarrollado en Portugal, tiene las siguiente ventajas:

* Voces de alta calidad; * Un excelente tiempo de respuesta; * Posibilidad para configurar variante, volumen y velocidad; * Permite la lectura a muy alta velocidad; * Más de 50 voces en casi 30 idiomas diferentes; * Puede utilizarse con NVDA portable; * El precio final de usuario incluye siempre, al menos, 10 EUROS para el desarrollo de NVDA *
