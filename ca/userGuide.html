1. Introducci&oacute;

NonVisual Desktop Access (NVDA) &eacute;s un lector de pantalla de codi obert gratuït per al sistema operatiu Microsoft Windows. NVDA permet a les persones cegues i amb baixa visi&oacute; accedir als ordinadors que funcionen amb Windows amb el mateix esforç que a una persona amb visi&oacute; gr&agrave;cies a proveir feedback amb veu sintètica i braille. NVDA ha estat desenvolupat per NV Access, amb contribucions de la comunitat.

1.1. Caracter&iacute;stiques generals

NVDA permet a les persones cegues i amb baixa visi&oacute; accedir i interaccionar amb el sistema operatiu Windows i amb m&uacute;ltiples aplicacions de tercers.

En s&oacute;n aspectes destacables:

  • NVDA &eacute;s compatible amb aplicacions comunes com navegadors web, clients de correu electrònic, programes de xat, i paquets d'ofim&agrave;tica
  • NVDA incorpora un sintetitzador de veu amb suport per a 43 idiomes
  • Si el programa d&oacute;na informaci&oacute; sobre el format del text, NVDA n'informa i anuncia la fam&iacute;lia tipogr&agrave;fica, el cos, l'estil i avisa d'errors ortogr&agrave;fics
  • NVDA avisa autom&agrave;ticament del text que es troba sota el ratol&iacute; i opcionalment indica la posici&oacute; del ratol&iacute; per so
  • NVDA &eacute;s compatible amb moltes l&iacute;nies braille, i tamb&eacute; accepta entrades en braille d'ordinador per aquelles l&iacute;nies braille que tenen teclat
  • NVDA pot executar-se totalment des d'un llapis USB o altres suports portables, sense requerir instal·laci&oacute;
  • L'instal·lador de NVDA &eacute;s amb veu i f&agrave;cil d'usar
  • NVDA ha estat traduït a 44 idiomes
  • NVDA &eacute;s compatible amb els sistemes operatius Windows moderns, tant amb les versions de 32 bits com les de 64 bits
  • NVDA funciona a la pantalla d'entrada de Windows i en altres pantalles de seguretat
  • NVDA &eacute;s compatible amb interf&iacute;cies d'accessibilitat comunes com Microsoft Active Accessibility, Java Access Bridge, IAccessible2 o UI Automation (NVDA nom&eacute;s &eacute;s compatible amb UI Automation a partir de Windows 7)
  • NVDA &eacute;s compatible amb la l&iacute;nia d'instruccions Windows i amb aplicacions de consola

A banda de facilitar els missatges i la interf&iacute;cie en m&uacute;ltiples idiomes, NVDA permet que l'usuari llegeixi el contingut en qualsevol idioma, sempre i quan tingui un sintetitzador de veu en aquell idioma.

Si vols portar NVDA amb tu en un llap&iacute;s USB o en algun altre suport d'escriptura, llavors hauries de triar de crear la versi&oacute; portable. Una altra opci&oacute; &eacute;s fer servir la versi&oacute; temporal de NVDA (per ex. amb finalitats demostratives), però iniciar NVDA d'aquesta manera cada vegada pot suposar molt de temps.

3.1. Restriccions en les versions portable i temporal

A banda de no poder iniciar-se autom&agrave;ticament durant o despr&eacute;s de l'entrada, les versions portables i temporals de NVDA tenen les següents restriccions:

  • No poden interaccionar amb aplicacions que s'executen amb privilegis d'administrador, a no ser que NVDA s'executi tamb&eacute; amb permisos d'administrador (no recomanable).
  • No poden llegir pantalles d'User Account Control (UAC) quan volen iniciar una aplicaci&oacute; amb privilegis d'administrador.
  • Windows 8: no s&oacute;n compatibles amb l'entrada des de pantalla t&agrave;ctil.
  • Windows 8: no ofereixen algunes funcions en les apps de la Windows Store, com ara el mode de navegaci&oacute; o la verbalitzaci&oacute; de car&agrave;cters escrits.

Aquesta opci&oacute; et permet triar si vols que NVDA s'inicii autom&agrave;ticament a l'inici de sessi&oacute; de Windows, abans que introdueixis la contrassenya. Això tamb&eacute; inclou controls UAC i altres pantalles de seguretat.

Quan NVDA comença per primer cop, el programa et saluda amb una caixa de di&agrave;leg que d&oacute;na informaci&oacute; b&agrave;sica sobre la tecla modificadora NVDA i el men&uacute; NVDA. La tercera casella permet controlar si aquest di&agrave;leg de benvinguda ha d'apareixer cada cop que NVDA s'inicia.

4.2. Les instruccions de teclat de NVDA

4.2.1. La tecla modificadora NVDA

La majoria d'instruccions de teclat espec&iacute;fiques de NVDA es basen en pr&eacute;mer una tecla particular anomenada tecla modificadora NVDA i alhora pr&eacute;mer una o m&eacute;s tecles addicionals. De forma predeterminada, tant la tecla Insert del teclat numèric, com la tecla Insert del teclat estès, funcionen com a tecles modificadores NVDA.

Si vols que una de les tecles modificadores NVDA realitzi la seva funci&oacute; original (per exemple si vols activar el Bloqueig de maj&uacute;scules mentre que la tecla de Bloqueig de maj&uacute;scules funciona com a tecla modificadora NVDA), pots pr&eacute;mer la tecla r&agrave;pidament dos cops.

4.2.2. Disposicions de teclat

NVDA s'ofereix actualment amb dos conjunts d'instruccions de teclat. Existeix una disposici&oacute; de teclat per ordinadors de sobretaula i una disposici&oacute; de teclat per ordinadors port&agrave;tils. De forma predeterminada NVDA est&agrave; configurat per usar el teclat d'ordinadors de sobretaula, però pots canviar al teclat per ordinadors port&agrave;tils en la Configuraci&oacute; de teclat, que es troba sota Preferències al men&uacute; NVDA.

La disposici&oacute; de teclat de sobretaula fa un &uacute;s extensiu del teclat numèric (amb el Bloqueig numèric desactivat). Tot i que molts port&agrave;tils no tenen un teclat numèric f&iacute;sic, alguns port&agrave;tils l'emulen apretant la tecla de funci&oacute; (FN) i apretant les lletres i n&uacute;meros de la part dreta del teclat (7 8 9 u i o j k l etc). Si el teu port&agrave;til no t&eacute; aquesta funci&oacute;, o no et permet desactivar el bloqueig de n&uacute;meros, el m&eacute;s recomanable &eacute;s que canvïis a la disposici&oacute; de teclat per port&agrave;tils.

4.3. Gestos t&agrave;ctils NVDA

Si executes NVDA en un ordinador amb pantalla t&agrave;ctil, sobre Windows 8 o superior, pots controlar NVDA directament des de la pantalla t&agrave;ctil. Mentre NVDA s'executa, totes les entrades t&agrave;ctils aniran directament a NVDA. En conseqüència, les accions que normalment es poden fer sense NVDA no funcionaran.

4.3.1. Exploraci&oacute; de la pantalla

L'acci&oacute; m&eacute;s b&agrave;sica que pots fer amb la pantalla t&agrave;ctil &eacute;s anunciar el control o el text des de qualsevol punt de la pantalla. Per fer-ho, posa el teu dit a qualsevol lloc de la pantalla. Pots mantenir el dit a la pantalla i moure'l amunt i avall per llegir altres controls i text per on passi el dit.

4.3.2. Gestos t&agrave;ctils

Quan m&eacute;s endavant es descriguin les instruccions NVDA, sovint s'indicar&agrave; quin gest t&agrave;ctil s'usa per activar aquella instrucci&oacute; amb la pantalla t&agrave;ctil. A continuaci&oacute; trobar&agrave;s algunes instruccions sobre com fer els diferents gestos t&agrave;ctils.

Cops
