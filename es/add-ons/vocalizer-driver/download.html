Voces de Vocalizer para NVDA

¡Bienvenido a la página de descarga de Vocalizer para NVDA!

Aquí puedes descargar un complemento para NVDA, con el controlador Vocalizer y varias voces disponibles.

Para instalar el controlador y las voces que decidas, realiza los siguientes pasos:

  1. Abre el menú NVDA, con NVDA+N;
  2. Pulsa flecha abajo hasta el submenú Herramientas y pulsa Intro para abrirlo;
  3. Busca el elemento "Administrador de Complementos" y pulsa Intro;
  4. Haz clic sobre el botón "Instalar" y en el siguiente diálogo examina para localizar el Complemento deseado;
  5. Después de pulsar el botón "Abrir", NVDA presentará un diálogo de aviso preguntándote para confirmar que quieres instalar. Responde sí para continuar;
  6. Ejecuta estos pasos hasta haber instalado todos los Complementos deseados. Para finalizar, Tabula hasta el botón "Cerrar" y pulsa Intro. Reinicia NVDA cuando se te indique.
